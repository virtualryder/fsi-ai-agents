# ROI Analysis — AML/TMS Enhancement Agent

## The False Positive Problem

Industry data (FinCEN, Celent, McKinsey):
- Average TMS false positive rate: **85–95%**
- Analyst cost to review one alert: **$25–40** (25-35 min at $80K fully-loaded)
- Mid-sized bank (500 alerts/day): **$4.3M–$7.3M/year wasted on noise**

## Conservative ROI Model (Mid-Sized Bank)

| Metric | Baseline | With AI Layer |
|--------|----------|---------------|
| Daily alerts | 500 | 500 |
| False positive rate | 90% | ~45% (50% reduction) |
| Alerts reaching analysts/day | 500 | ~275 |
| Analyst hours/day on FP review | 225h | ~112h |
| Annual FP review cost | $4.1M | $2.0M |
| **Annual savings** | — | **$2.1M** |

AI layer annual cost (infrastructure + licensing): ~$180K–$280K
**Net annual benefit: $1.8M–$2.0M**
**ROI: 6–11x**

## Quality Improvements

Beyond cost savings:
- **Analyst focus**: Remaining alerts are higher-quality (lower FP rate = higher SAR rate)
- **Escalation speed**: Genuine high-risk alerts are fast-tracked before they reach the queue
- **Regulatory posture**: Explainable suppression records satisfy SR 11-7 model governance
- **Examiner confidence**: Full audit trail for every suppression decision

## Combined Pitch (with Investigation Agent)

| Product | Value Proposition |
|---------|-------------------|
| Financial Crime Investigation Agent | Investigators work 3x faster when alerts arrive |
| AML/TMS Enhancement Agent | 50% fewer alerts arrive in the first place |
| **Combined** | **Lower volume AND faster investigation = maximum analyst efficiency** |

A bank running both agents:
- Reduces total alert volume by ~50%
- Cuts investigation time per surviving alert by 65%
- Net effect: same analyst headcount handles 3x the compliance workload
